# Network Security Labs
Hands-on Network Security labs using GNS3, Wireshark, and Kali Linux.

-----
## Overview

This repository contains my hands-on Network Security labs completed during my MS Cybersecurity studies.

The labs were performed using GNS3, Wireshark, Kali Linux, and Ubuntu Linux to strengthen my practical knowledge of networking and network security.

I created this repository to document my learning journey, organize my lab work, and share practical knowledge with students and anyone interested in Network Security.

-----

## Objectives

- Document my Network Security learning journey.
- Organize hands-on labs in one place.
- Practice and improve practical networking skills.
- Share technical knowledge with students and the cybersecurity community.
- Build a professional cybersecurity portfolio.

- Wireshark Packet Analysis

  ## Learning Outcomes

After exploring these labs, readers will be able to:

- Understand fundamental Network Security concepts.
- Build and configure secure network topologies.
- Analyze network traffic using Wireshark.
- Configure routing and switching in GNS3.
- Practice network troubleshooting techniques.
- Gain hands-on experience with common networking tools and technologies.
